# Native PHP Login System

A **secure login system** built using **native PHP**, following **MVC architecture** and best practices. Ideal for learning or integrating into local web applications.

## Features

- ✅ User Registration & Login
- ✅ Password Hashing (`password_hash`)
- ✅ CSRF Protection
- ✅ “Remember Me” Persistent Login
- ✅ Protected Dashboard
- ✅ Logout Functionality
- ✅ Bootstrap 5 Frontend for Responsive Design
- ✅ MVC Structure (Models, Views, Controllers)

## Technology Stack

- PHP 8+ (native, no frameworks)
- MySQL / MariaDB
- PDO for database interaction
- Bootstrap 5 for styling
